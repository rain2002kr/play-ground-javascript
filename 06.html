배열 Array

1. 배열 만드는 방법

  • obj1 = [];

    obj2 = new Array();

2. push, pop, unshift, shift, splice 펑션 Array 인터페이스 함수

  • push : array 끝에 data 추가

    pop : array 끝에 data 삭제

    unshift : array 앞에 data 추가

    shift : array 앞에 data 삭제

    splice : array ( 시작점, 수량, 추가분 )

3. concat, set, join, slice, sort, reverse, indexOf, forEach, filter, map ,flat

  • 오브젝트 할당을 함수로 하는 방법

    매개변수와 리턴값이 같으면, 생략이 가능

    함수에 기능이 없다면, 생성자형으로 만드는것이 일반적인 관례임.

    new 키워드와 대문자, 콘솔로 찍으면 이름이 나옴. (이 방식 추천)

4. in operator 키의 유무를 확인

  • console.log('name' in khlee)

    키의 이뮤를 확인하고 true, false 값을 리턴해준다.

5. 반복 출력 for in 오브젝트 and for of 배열

  • 오브젝트를 순회 하는 for in

    배열을 순회하는 for of

    복합형은 추후 확인
