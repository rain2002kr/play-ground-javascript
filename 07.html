3. AJAX

  • 웹페이지에서 동적으로 서버로 데이터를 주고 받을수 있는 기술을 의미한다

4. XMLHttpRequest XHR

  • XMLHttpRequest API 를 이용하여 통신 가능.

    약자로 XHR 오브젝트

    브라우저 API에서 제공하는 오브젝트중의 하나임

    이 오브젝트를 이용하면, 간단하게 서버에 데이터를 요청하고 받아올수가 있다.

    XML 은 파일 커지고 가독성 안좋아서 요즘 많이 사용 안함.

5. FETCH API

  • 최근 브라우저에 추가된 최신 API 임 (내장 API)

    인터넷 익스플로러에 제공안됨

6. XHR 이름 관련 개발 이야기

  • 마이크로소프트 아웃룩팀이 AJAX 와 XHR 통신 관련 개발에 적극 참여

    이때 XML 파일포멧을 사용했는데 XHR 은 XML 데이터만 주고받는게 아니지만

    HttpRequest 앞에 XML 에 따서 XMLHttpRequest 로 이름을 짓고, 약자로 XHR 이라고함
