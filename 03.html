function 펑션

펑션

1. 펑션 선언

  • 이름 짓는법 : doSomething, command, verb

    이름이 한가지로 정의 안될때는 함수 쪼개기

2. 타입 스크립트

  • TypeScript playGround 에서 익혀 보기

    큰 프로젝트를 하는 경우는 타입스크립트 필요

3. 함수 파라메터 primitive vs object

  • primitive

    passed by value 값이 전달

    object

    passed by reference 주소값 전달

4. Default 파라메터

  • ES6 에 추가

    파라메터에 값이 전달되지 않을때, Default 로 선언된 값이 들어감

    function showMessage(message, from = 'default')

5. rest parameter ...

  • function printAll(...args){

    for(const arg of args){ console.log(arg); }

    args.forEach((arg) => console.log(arg))

6. 로컬 스코프

  • 클로저 : 부모함수 에서 자식함수 안에 접근 가능

    스코프 개념 : 밖에서는 안이 보이지 않고 안에서만 밖을 볼수 있다.

    let global = "show";

    function local (){ let local = "in";

    console.log(local); //접근 안됨

7. 코드 팁

  • 조건이 맞이 않으면 함수를 종료하고 조건이 맞을때만 함수 로직 실행

8. 펑션 기능 : 이름이 있는 경우 vs 없는 경우

  • const fc = function (){return false;}

    const fc = function named(){return false;}

    디버깅 할때 스택트레이스에 쌓이기 때문에 named 펑션을 쓰는것이 좋다.

9. 콜백 함수 : 펑션을 인자로 사용 가능

  • function randomQuiz(answer, printYes, printNo){

    if (answer === 'love you'){ printYes();}

    else { printNo();}

10. 화살표 함수 : 애로우 펑션

  • const add = (인자) => { 리턴 }

    const add = (a, b) => { a + b }

11. 즉시 함수 : IIFE Immediately Invoked Function Express

  • ( function(){ } )();

    위처럼 함수 감싸고, 바로 호출

    ( ()=>console.log("test") ),()

12. 생성자 함수 : 오브젝트와 함께 사용

  • function Person(name,age) {return name, age}

    함수를 돌려주고 싶다면, 변수에 함수를 담아서, return 값에 추가 해줌

    function Person(name,age) {const fc = ()=>{log('something')}; return name, age, fc}
