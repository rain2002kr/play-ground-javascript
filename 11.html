비구조화 할당 & spread and rest 문법

1. 비구조화 할당.

  • 비구조화 할당은 오브젝트 및 배열의 요소를 분해하는데 사용 한다.

    const person = {name:"이경훈", age:39}

    const {name, age} = person

    위와 같이 간단하게 사용

    중첩 구조라고 해도 몇번의 분해로 바로 인자값들을 가져올수 있다.

2. spread 문법

  • spread 문법은 객체 혹은 배열에 값을 그대로 가져와서 새로운 객체 혹은 배열을 사용할때 사용한다.

    const tbs = {type:"machine",name:"abs"}

    const tbs = {...tbs, addition:4}

    const arr = [1,2,3,4,5]

    const [one,two, ...rest] = arr

3. rest 문법

  • rest 문법은 객체 혹은 배열에 값을, 분해할당으로 가져가고 나머지를 가져오는데 사용된다.

    const tbs = {type:"machine",name:"abs",addition:4}

    const {type, ...rest} = tbs

    이때 rest 는 분해할당과 다르게 이름을 바꿀수 있다.
