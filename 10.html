Async and Promise

1. Promise 를 좀더 사용하기 쉽게 래핑한 Async 에 대해 알아보자.

  • Async 는 Promise 의 보일러코드를 자동으로 변환 해준다.

    Async 를 선언해 줌으로써, 알아서 코드가 return new Promise...로 변환 되는 구조 이다.

    다음 코드는 동일하다. new Promise 와 resolve 와 reject 를 선언 하지 않아도 된다.

    function login(id,pass){ return new Promise(resolve,reject) { if(id=="khlee){ resolve(id)} else { reject(new Error("not found")) };

    async function login(id,pass){ return { if(id=="khlee){ id } else { new Error("not found") };

    try catch(err) 구문으로 error 처리가 가능하다.

2. Promise 내장 함수 all

  • Promise.all 함수

    병렬처리된 함수의 결과값을 배열로 리턴 합니다.

    Promise.all([함수1,함수2])

3. Promise 내장 함수 race

  • Promise.race 함수

    병렬처리된 함수의 결과값중 가장 먼저 처리되는 결과값을 리턴 합니다.

    Promise.race([함수1,함수2])
